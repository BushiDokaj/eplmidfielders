Data Dictionary

Statistics

Mean:

The average value of the given data

 Max:

The greatest value in a given set of data points.

 90% CI:

A confidence interval. We can be 90% certain, for the given data category, that the player will score within this range.

Data

Year:

The Premier League season.

 Appearances:

The number of games a player featured in that season.

 Goals:

The number of goals a player scores in a season.

 Goals Per Match:

The proportion of goals the player scored to the number of matches they played that season.

 Assists:

The number of assists a player records in a season.

 Assists Per Match:

The proportion of assists to the number of matches played.

 Goal Contributions:

The total number of goals plus assists recorded by a player in a season.

 Goal Contributions Per Match:

The proportion of goal contribtions to the number of matches played.

 Shots:

The total number of shots taken by a player in a season. This data is not available prior to the 2006/07 season.

 Shots on Target:

The total number of shots that taken that were on goal. This data is not available prior to the 2006/07 season.

 Shot Accuracy:

The percentage of shots a player takes that are on goal. This data is not available prior to the 2006/07 season.

 Shot Conversion Rate:

The percentage of shots a player takes that lead to a goal. This data is not available prior to the 2006/07 season.

 Passes:

The total number of passes made by a player in a season. This data is not available prior to the 2006/07 season.

 Passes Per Match:

The proportion of passes to the number of matches played. This data is not available prior to the 2006/07 season.

 Big Chances Created:

The number of big chances a player created that season. This data is not available prior to the 2010/11 season.

 Total Crosses:

The total number of crosses made by a player in a season. This data is not available prior to the 2006/07 season.

 Cross Accuracy:

The percentage of crosses made by a player that find their target. This data is not available prior to the 2006/07 season.

 Through Balls:

The total number of through balls made by a player. This data is not available prior to the 2007/08 season.

 Accurate Long Balls:

The total number of accurate long balls made by a player. This data is not available prior to the 2006/07 season.

 Tackles:

The total number of tackles a player makes in a season. This data is not available prior to the 2006/07 season.

 Tackle Success:

The percentage of tackles a player makes that are successful. This data is not available prior to the 2006/07 season.

 Shots Blocked:

The total number of shots blocked by a player. This data is not available prior to the 2006/07 season.

 Interceptions:

The total number of interceptions made by a player. This data is not available prior to the 2006/07 season.

 Clearances:

The total number of clearances made by a player. This data is not available prior to the 2006/07 season.

 Recoveries:

The total number of recoveries made by a player. This data is not available prior to the 2007/08 season.

 Duels Won:

The total number of 1 on 1 duels won by a player. This data is not available prior to the 2006/07 season.

 Duels Lost:

The total number of 1 on 1 duels lost by a player. This data is not available prior to the 2006/07 season.

 Successful 50/50s:

The total number of 50/50 balls won by a player. This data is not available prior to the 2006/07 season.

 Aeiral Battles Won:

The number of headers won by a player. This data is not available prior to the 2006/07 season.

 Aeiral Battles Lost:

The number of headers lost by a player. This data is not available prior to the 2006/07 season.

 Errors Leading to Goal:

The number of goals conceeded as a direct result of an error made by a player.
